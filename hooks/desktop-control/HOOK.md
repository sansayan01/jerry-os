# Desktop Control

Provides pixel-perfect mouse control, keyboard input, screen capture, and window management.

## Features

- Mouse movement and clicking
- Keyboard typing and hotkeys
- Screenshots
- Window management

## Commands

- "click at 500 300" - Click at coordinates
- "type hello world" - Type text
- "screenshot" - Capture screen
- "move mouse to 100 200" - Move cursor
- "press ctrl c" - Keyboard shortcut
